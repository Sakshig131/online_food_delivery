# 🍔 Foodie Hub - Online Food Delivery Website

## 📌 Project Description
Foodie Hub is a simple and user-friendly **Online Food Delivery Website**.  
It allows users to browse food items, add them to cart, and place orders easily.

This project is developed as a mini project using basic web technologies.

---

## 🎯 Features

- 🍽️ Browse food menu (Burger, Pizza, Pasta)
- 🛒 Add to Cart functionality
- 📦 Cart page with total calculation
- 💳 Checkout system
- 🚀 Delivery time display (30–40 mins)
- 📍 Embedded Google Map
- 📱 Responsive design using Bootstrap
- ✅ Order confirmation alert

## ⚙️ How to Run the Project

1. Download or clone the repository  
2. Open the project folder in VS Code  
3. Right click on `index.html` → Click **Open with Live Server**  

👉 Project will run in browser like this:  
http://127.0.0.1:5500  

👉 Or manually:
- Open `index.html`
- Double click OR open with browser 

---

## 🔁 Working Flow

1. User opens Home Page  
2. Clicks on "Order Now"  
3. Selects food items from Menu  
4. Adds items to Cart  
5. Views Cart and total price  
6. Proceeds to Checkout  
7. Enters details and confirms order
